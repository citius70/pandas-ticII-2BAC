Ejercicios de visualización de datos#

En este notebook aprenderás a crear diferentes tipos de gráficos utilizando solo pandas. No necesitaremos importar bibliotecas adicionales como matplotlib o seaborn, ya que pandas permite crear gráficos directamente desde DataFrames y Series.

import pandas as pd

# Cargar datos de ejemplo
url = "https://drive.google.com/uc?export=download&id=1HbZ6KvPgjnGGwi74lkHUr8bwyHd_v_42"
df = pd.read_csv(url)
