6. Visualizacion de datos en Pandas: Gráficos básicos#

Aunque Pandas no es una biblioteca de visualización avanzada como Seaborn o Plotly, permite generar gráficos rápidamente para analizar datos de manera exploratoria.

La biblioteca Pandas en Python tiene capacidades básicas de visualización mediante su integración con Matplotlib.

6.1. Tipos de gráficos que se pueden hacer con Pandas#

La función principal para graficar en Pandas es .plot(), que se basa en Matplotlib. Algunos de los gráficos más comunes son los que se muestran después del siguiente bloque:

# Creamos el dataframe df_artistas
import pandas as pd
df_artistas = pd.read_excel("https://docs.google.com/spreadsheets/d/15pDi5KoAatuUJ1o8hcD8Lgd6zqcW_SSM/export?format=xlsx")
df_artistas
Artista Spotify streams (millones)
0 Taylor Swift 29100
1 Bad Bunny 16370
2 The Weeknd 14140
3 Drake 14030
4 Peso Pluma 10540
5 Feid 9242

Nota: En general, en el eje x suele ir la etiqueta, y en el eje y el dato.

- Gráfico de barras verticales#

La sintaxis básica para dibujar un gráfico de barras es: plot.bar(x,y)

df_artistas.plot.bar(
    x="Artista", 
    y="Spotify streams (millones)"
)

<Axes: xlabel='Artista'>

- Grafico de barras horizontales#

Sintaxis básica: plot.barh(x,y)

df_artistas.plot.barh(
    x="Artista", 
    y="Spotify streams (millones)", 
    title="Reproducciones anuales", 
    color="green")

<Axes: title={'center': 'Reproducciones anuales'}, ylabel='Artista'>

- Gráfico de línea#

Sintaxis: plot.bar(x,y, kind='line')

df_artistas.plot(
    x='Artista',
    y='Spotify streams (millones)',
    kind='line',
    marker='x',
    color='gold',
    title="Reproducciones anuales",
    figsize=(6, 3)
)

<Axes: title={'center': 'Reproducciones anuales'}, xlabel='Artista'>

- Gráfico circular#

Sintaxis básica: plot.pie(x, y, labels, legend)

Para este gráfico añadiremos dos argumentos

  • labels: Este argumento especifica las etiquetas que se mostrarán.

  • legend: Este argumento controla si se mostrará o no la leyenda en el gráfico. La leyenda generalmente proporciona información sobre el significado de los colores o las porciones en el gráfico. En este caso si la activas bloquea un poco la información.

# En este caso vamos a utilizar colores elegidos por nosotros.
# Creamos un diccionario con ellos.
mis_colores = ["gold", "lightcoral", "skyblue", "palegreen", "orchid", "lightsalmon"]

# Creamos el gráfico
df_artistas.plot.pie(
    x="Artista", 
    y="Spotify streams (millones)", 
    title="Distribución de streams por artista",
    labels=df_artistas.Artista,
    autopct='%1.1f%%', # muestra porcentajes con un decimal
    colors=mis_colores,  # utilizamos los colores elegidos
    legend= False,
)

<Axes: title={'center': 'Distribución de streams por artista'}, ylabel='Spotify streams (millones)'>
