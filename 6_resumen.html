Resumen#

Pandas es una de las bibliotecas más importantes de Python para análisis y manipulación de datos. Está diseñada para trabajar con datos estructurados (como tablas) de manera eficiente y fácil. Aquí te explico sus principales funcionalidades:

1. Estructuras de datos principales#

  • Series:

    • Un array unidimensional (como una columna de Excel).

    • Ejemplo:

      import pandas as pd
datos = pd.Series([10, 20, 30], name="Edad")

  • DataFrame:

    • Una tabla bidimensional (como una hoja de Excel o SQL).

    • Ejemplo:

      datos = pd.DataFrame({
    "Nombre": ["Ana", "Juan", "Luis"],
    "Edad": [25, 30, 35]
})

2. Funcionalidades clave#

📊 Carga y guardado de datos#

  • Lee/escribe desde múltiples formatos:

    # CSV, Excel, JSON, SQL, etc.
df = pd.read_csv("datos.csv")
df.to_excel("salida.xlsx")

🔍 Manipulación de datos#

  • Filtrar datos:

    df[df["Edad"] > 30]  # Personas mayores de 30 años.

  • Agregar/eliminar columnas:

    df["Ciudad"] = ["Madrid", "Barcelona", "Valencia"]  # Añadir columna.
df.drop("Edad", axis=1, inplace=True)  # Eliminar columna.

📈 Operaciones estadísticas#

  • Resúmenes rápidos:

    df.describe()  # Media, percentiles, etc.
df["Edad"].mean()  # Promedio de edades.

🔄 Limpieza de datos#

  • Manejo de valores nulos:

    df.dropna()  # Elimina filas con valores faltantes.
df.fillna(0)  # Rellena con ceros.

  • Eliminar duplicados:

    df.drop_duplicates()

📊 Agrupación y agregación#

  • Agrupar por categorías:

    df.groupby("Ciudad")["Edad"].mean()  # Edad promedio por ciudad.

⏳ Manejo de series temporales#

  • Fechas como índice:

    df["Fecha"] = pd.to_datetime(df["Fecha"])
df.set_index("Fecha", inplace=True)

🔗 Combinación de datos#

  • Unir tablas (como SQL):

    pd.merge(df1, df2, on="ID")  # Inner join.

3. Visualización (integrado con Matplotlib/Seaborn)#

df["Edad"].plot(kind="hist")  # Histograma.
df.plot.scatter(x="Edad", y="Ingresos")  # Gráfico de dispersión.

4. Rendimiento optimizado#

  • Operaciones vectorizadas: Más rápidas que los bucles for.

  • Compatibilidad con big data: Integración con Dask o modin para datasets grandes.

Ejemplo completo#

import pandas as pd

# Cargar datos
datos = pd.read_csv("clientes.csv")

# Limpiar y transformar
datos = datos.dropna()
datos["Ingresos_Anuales"] = datos["Ventas"] * 0.30  # Nueva columna.

# Análisis
print(datos.groupby("Región")["Ingresos_Anuales"].sum())

# Guardar
datos.to_excel("resultados.xlsx")

📌 ¿Para qué se usa Pandas en proyectos reales?#

  • Ciencia de datos: Preprocesamiento para machine learning.

  • Finanzas: Análisis de series temporales.

  • Web scraping: Organizar datos extraídos.

  • Automoción: Procesamiento de datos de sensores.

Es una herramienta imprescindible si trabajas con datos en Python. 🚀
