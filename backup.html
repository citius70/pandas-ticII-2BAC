Un problema com√∫n la hora de gestionar datos es que hay columnas que no tienen todos los datos. Miremos este ejemplo:

Advertencia: el m√©todo dropna() elimina toda la fila si es que alguna columna es NaN.

Nota: Tienes que crear otro DataFrame con la informaci√≥n filtrada. Esto es una protecci√≥n para evitar que elimines f√°cilmente los datos.

Otra manera de gestionar estos datos es llenando el vacio que dej√≥. Vamos a reemplazar el NaN en este caso por 7900, que fue el n√∫mero asociado a Feid para este a√±o, usando fillna
